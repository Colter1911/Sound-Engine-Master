# SEM (Sound Engine Master)

## Основное описание

**SEM (Sound Engine Master)** — модуль для Foundry VTT, который заменяет стандартную работу со звуком.

Если вы пользовались встроенными плейлистами Foundry, вы знаете их ограничения: нет нормального управления каналами, файлы добавляются по одному, поиск нужного трека во время сессии — отдельное приключение. SEM — это полнофункциональный аудио-инструмент, которым удобно пользоваться прямо за столом.

- **Движок.** Собственный аудио-движок, работающий параллельно Audio API Foundry. Качественное, плотное звучание, которого стандартными средствами Foundry не добиться. Устраняет заикания на тяжёлых треках, не грузит сеть в простое. Синхронизация устроена так, что игроки гарантированно слышат то же, что и ГМ.

- **Микшер.** Полноценное канальное управление: фейдеры, списки воспроизведения, перемотка. Для каждой игры можно заранее собрать очереди — эмбиент, музыка, SFX — и переключать их в одно действие, а не искать нужный файл посреди боя. Режим Sync OFF позволяет прослушивать треки у себя в наушниках и подбирать момент для вступления, не транслируя звук игрокам раньше времени.

- **Импорт и организация.** Drag-n-drop папками прямо из проводника — 10, 50, 100 файлов за раз. При импорте можно выставить теги, и они автоматически присвоятся всем добавленным трекам. Перетаскивание треков по плейлистам, организация тегов. Организация библиотеки, которая раньше занимала часы - укладывается в минуты.

- **Миграция.** Существующие плейлисты из стандартного Foundry переносятся перетаскиванием. SEM не просто копирует ссылки — он конвертирует всё в свою базу с глобальным поиском, фильтрацией и обработкой. Ничего не теряется, только приобретается в качестве и удобстве.

- **Эффекты.** Обработка звука в реальном времени. Есть готовые пресеты под типовые ситуации — «Cathedral» (ревер + задержка), «Underwater» (фильтр низких частот) — или можно собрать свою цепочку из модулей: компрессор, дисторшн, эквалайзер и другие.

- **Библиотека.** Поиск по тегам (`#Horror #Tension #Rain`) вместо имён файлов. Глобальный поиск по всей коллекции, фильтрация по каналам, подсветка активных плейлистов. Вы всегда знаете, где лежит нужный звук.
